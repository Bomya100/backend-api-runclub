# 🏃‍♂️ Run Club API

ระบบ Backend Service ที่ออกแบบมาเพื่อบันทึกสถิติการวิ่ง จัดการแผนการซ้อม และเก็บข้อมูลประสิทธิภาพสำหรับนักวิ่งระยะไกลโดยเฉพาะ

## 🚀 เทคโนโลยีที่ใช้ (Tech Stack)
* **Runtime:** Node.js
* **Database:** MongoDB
* **Infrastructure:** Docker & Docker Compose

## ✨ ฟีเจอร์หลัก
* โครงสร้างแบบ RESTful API สำหรับจัดการข้อมูลผู้ใช้งานและประวัติการวิ่ง
* ติดตามและบันทึกค่าสถิติที่สำคัญ เช่น เพซ (Pace), โซนอัตราการเต้นของหัวใจ (Heart Rate Zones) และระยะทาง
* โครงสร้างข้อมูลรองรับการวางแผนตารางซ้อม
* จัดการสภาพแวดล้อมการทำงานด้วย Container (Docker) เพื่อให้ติดตั้งและนำไปใช้งานจริงได้ง่าย

## 🛠️ วิธีการติดตั้งและรันระบบ

1. โคลน (Clone) โปรเจกต์นี้ลงมาที่เครื่องคอมพิวเตอร์ของคุณ:
   ```bash
   git clone https://github.com/Bomya100/backend-api-runclub.git

2. สร้างไฟล์ .env ไว้ที่โฟลเดอร์หลักของโปรเจกต์ และกำหนดค่าตัวแปรแวดล้อมให้เรียบร้อย

3. สั่งรันระบบด้วย Docker Compose:
    ```bash
    docker compose up -d

4. เซิร์ฟเวอร์ API จะพร้อมให้เรียกใช้งานที่ `http://localhost:5000`
